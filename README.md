- **🌐 Cross-Platform** - Works seamlessly across all modern browsers and devices

## 🛠️ Local Development

### Environment Setup
```bash
# Clone the repository
git clone https://github.com/michaelgermini/Complete-ECharts-mastery-guide.git

# Navigate to project directory
cd Complete-ECharts-mastery-guide

# Install dependencies (optional)
npm install

# Start local development server
npm start
```

### Development Scripts
```bash
npm start          # Start local development server
npm run dev        # Development mode with live reload
npm run build      # Build for production
npm test           # Run tests
npm run lint       # Lint code
npm run format     # Format code
```

### Alternative Local Server

## 🌐 Deployment

### GitHub Pages (Recommended)

#### Automatic Deployment
1. **Push to main branch** - GitHub Actions automatically deploys
2. **Enable GitHub Pages** - Go to repository Settings > Pages
3. **Select source** - Choose "gh-pages" branch
4. **Access your site** - Available at `https://username.github.io/repository-name`

#### Manual Setup
```bash
# Create gh-pages branch
git checkout -b gh-pages

# Add all files
git add .

# Commit changes
git commit -m "Deploy to GitHub Pages"

# Push to GitHub
git push origin gh-pages
```
