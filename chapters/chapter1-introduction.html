üìä Chapter 1: What is ECharts?

Your first steps into the world of data visualization with Apache ECharts

Introduction to Apache ECharts

Apache ECharts is a powerful JavaScript library for building interactive, highly customizable charts and visualizations. It is developed under the Apache Software Foundation, and it's widely used in dashboards, analytics platforms, and even large-scale smart city projects.

üéØ Why Choose ECharts?

Other chart libraries like Chart.js or D3.js are great, but ECharts brings some unique strengths:

  • Rich library of chart types ‚Äì over 20 built-in chart types
  • Interactive & animated ‚Äì tooltips, zoom, drill-downs, brushing
  • Handles big data ‚Äì optimized for large-scale datasets
  • Easy setup ‚Äì works with just a single script tag
  • Cross-platform ‚Äì works in browsers, mobile apps, and embedded systems

Your First ECharts Chart

Let's create your very first chart! Below is a simple bar chart that demonstrates the basic ECharts setup:

Interactive Bar Chart Example

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First ECharts</title>
  <style>
    #main {
      width: 600px;
      height: 400px;
    }
  </style>
</head>
<body>
  <!-- Chart container -->
  <div id="main"></div>

  <!-- Import ECharts via CDN -->
  <script src="https://cdn.jsdelivr.net/npm/echarts/dist/echarts.min.js"></script>
  <script>
    // Initialize the chart
    var chart = echarts.init(document.getElementById('main'));

    // Define chart options
    var option = {
      title: {
        text: 'Hello ECharts'
      },
      tooltip: {},
      xAxis: {
        data: ['A', 'B', 'C', 'D', 'E']
      },
      yAxis: {},
      series: [{
        name: 'Example',
        type: 'bar',
        data: [5, 20, 36, 10, 15]
      }]
    };

    // Apply the options
    chart.setOption(option);
  </script>
</body>
</html>

Anatomy of an ECharts Option

Every ECharts visualization is defined by an option object. Here are the main components:

Chart Components Breakdown

var option = {
  title: { text: 'Sales Overview' },        // Chart title
  tooltip: {},                              // Interactive hover info
  legend: { data: ['Sales'] },              // Categories of data series
  xAxis: { data: ['Jan', 'Feb', 'Mar'] },   // X-axis configuration
  yAxis: {},                                // Y-axis configuration
  series: [{                                // Data and chart type
    name: 'Sales',
    type: 'line',
    data: [150, 230, 224]
  }]
};

üéØ Mini-Exercise

Try these modifications to practice:

Hint: Look for the type: 'bar' and text: 'Hello ECharts' in the code above!

üí° Pro Tips

  • ‚úÖ Always set a fixed width and height on your chart container, otherwise the chart won't show up
  • ‚úÖ Use echarts.init() only once per container
  • ‚úÖ The chart container must exist in the DOM before initializing ECharts
  • ‚úÖ Use chart.setOption() to update chart data dynamically
