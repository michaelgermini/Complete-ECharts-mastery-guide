Line Charts: The Foundation of Time Series

Line charts are perfect for showing trends over time, continuous data, and relationships between variables. They're one of the most commonly used chart types in data visualization.

🎯 When to Use Line Charts

  • Time series data - Stock prices, temperature, sales over time
  • Continuous data - Measurements, sensor readings
  • Trend analysis - Growth patterns, seasonal variations
  • Multiple series comparison - Different metrics over the same time period

1. Basic Line Chart

Simple Line Chart Example

var option = {
  title: { text: 'Monthly Visitors' },
  xAxis: { data: ['Jan', 'Feb', 'Mar', 'Apr', 'May'] },
  yAxis: {},
  series: [{
    type: 'line',
    data: [820, 932, 901, 934, 1290]
  }]
};

2. Multiple Lines

Compare multiple data series on the same chart:

Multiple Line Series

var option = {
  title: { text: 'Website Traffic Sources' },
  tooltip: { trigger: 'axis' },
  legend: { data: ['Desktop', 'Mobile', 'Tablet'] },
  xAxis: { data: ['Mon', 'Tue', 'Wed', 'Thu', 'Fri'] },
  yAxis: {},
  series: [
    { name: 'Desktop', type: 'line', data: [120, 132, 101, 134, 90] },
    { name: 'Mobile', type: 'line', data: [220, 182, 191, 234, 290] },
    { name: 'Tablet', type: 'line', data: [150, 232, 201, 154, 190] }
  ]
};

3. Line Styling and Customization

Customized Line Styles

series: [{
  name: 'Sales',
  type: 'line',
  data: [120, 200, 150, 80, 70, 110],
  smooth: true,                    // Smooth curves
  lineStyle: {                     // Line styling
    width: 3,
    color: '#4a90e2'
  },
  itemStyle: {                     // Point styling
    color: '#4a90e2',
    borderWidth: 2,
    borderColor: '#fff'
  },
  areaStyle: {                     // Area fill (optional)
    color: {
      type: 'linear',
      x: 0, y: 0, x2: 0, y2: 1,
      colorStops: [{
        offset: 0, color: 'rgba(74, 144, 226, 0.3)'
      }, {
        offset: 1, color: 'rgba(74, 144, 226, 0.1)'
      }]
    }
  }
}]

Area Charts: Adding Depth to Trends

Area charts are line charts with the area below the line filled. They're great for showing volume, cumulative data, and part-to-whole relationships.

4. Basic Area Chart

Simple Area Chart

var option = {
  title: { text: 'Rainfall Over Time' },
  xAxis: { data: ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun'] },
  yAxis: { name: 'Rainfall (mm)' },
  series: [{
    type: 'line',
    areaStyle: {},                 // Enable area fill
    data: [20, 40, 30, 50, 60, 45]
  }]
};

5. Stacked Area Chart

Show how different components contribute to a total over time:

Stacked Area Chart

var option = {
  title: { text: 'Revenue by Product Category' },
  tooltip: { trigger: 'axis' },
  legend: { data: ['Electronics', 'Clothing', 'Books'] },
  xAxis: { data: ['Q1', 'Q2', 'Q3', 'Q4'] },
  yAxis: {},
  series: [
    {
      name: 'Electronics',
      type: 'line',
      stack: 'total',              // Enable stacking
      areaStyle: {},
      data: [120, 132, 101, 134]
    },
    {
      name: 'Clothing',
      type: 'line',
      stack: 'total',
      areaStyle: {},
      data: [220, 182, 191, 234]
    },
    {
      name: 'Books',
      type: 'line',
      stack: 'total',
      areaStyle: {},
      data: [150, 232, 201, 154]
    }
  ]
};

6. Advanced Line Chart Features

Step Line Chart

Smooth Line Chart

// Step line chart
series: [{
  type: 'line',
  step: 'start',                   // 'start', 'middle', 'end'
  data: [120, 200, 150, 80, 70, 110]
}]

// Smooth line chart
series: [{
  type: 'line',
  smooth: true,                    // Enable smooth curves
  data: [120, 200, 150, 80, 70, 110]
}]

7. Time Series with Real Dates

Time Series Chart

var option = {
  title: { text: 'Stock Price Over Time' },
  tooltip: { trigger: 'axis' },
  xAxis: {
    type: 'time',                  // Time axis
    axisLabel: {
      formatter: '{MM-dd}'         // Date format
    }
  },
  yAxis: {},
  series: [{
    type: 'line',
    data: [
      ['2024-01-01', 100],
      ['2024-01-02', 105],
      ['2024-01-03', 98],
      ['2024-01-04', 110],
      ['2024-01-05', 108]
    ]
  }]
};

8. Interactive Features

Interactive Line Chart with Zoom

var option = {
  title: { text: 'Interactive Line Chart' },
  tooltip: {
    trigger: 'axis',
    axisPointer: {
      type: 'cross'                // Cross-hair pointer
    }
  },
  dataZoom: [{                     // Zoom control
    type: 'inside',
    start: 0,
    end: 100
  }, {
    type: 'slider',
    start: 0,
    end: 100
  }],
  xAxis: { data: ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun'] },
  yAxis: {},
  series: [{
    type: 'line',
    data: [120, 200, 150, 80, 70, 110]
  }]
};

💡 Pro Tips for Line Charts

  • Use smooth lines for better visual appeal with fewer data points
  • Limit series to 3-5 lines to avoid clutter
  • Use different line styles (solid, dashed, dotted) to distinguish series
  • Add area fills to emphasize volume or cumulative data
  • Include data points for precise value reading
