7.1 Introduction to Radar Charts

Radar charts (also known as spider charts or star charts) are excellent for comparing multiple variables across different categories. They display data on axes radiating from a central point, making it easy to see patterns and identify strengths and weaknesses.

When to Use Radar Charts:

  • Comparing multiple variables for different entities (e.g., product features, employee skills)
  • Showing performance across different dimensions
  • Identifying strengths and weaknesses in a dataset
  • Comparing multiple datasets with the same variables

7.2 Basic Radar Chart

Let's start with a simple radar chart showing student performance across different subjects.

Student Performance by Subject


var basicRadarChart = echarts.init(document.getElementById('basic-radar-chart'));
var basicRadarOption = {
    title: { text: 'Student Performance by Subject', left: 'center', textStyle: { color: '#333' } },
    tooltip: {
        trigger: 'item',
        formatter: '{b}: {c}%'
    },
    legend: {
        data: ['John', 'Sarah', 'Mike'],
        bottom: 0
    },
    radar: {
        indicator: [
            { name: 'Math', max: 100 },
            { name: 'Science', max: 100 },
            { name: 'English', max: 100 },
            { name: 'History', max: 100 },
            { name: 'Art', max: 100 },
            { name: 'Physical Education', max: 100 }
        ],
        radius: '65%',
        center: ['50%', '50%'],
        shape: 'polygon',
        axisName: {
            color: '#333',
            fontSize: 12
        },
        splitLine: {
            lineStyle: {
                color: ['#ddd']
            }
        },
        splitArea: {
            show: true,
            areaStyle: {
                color: ['rgba(250,250,250,0.3)', 'rgba(200,200,200,0.3)']
            }
        }
    },
    series: [{
        name: 'Student Performance',
        type: 'radar',
        data: [
            {
                value: [85, 90, 75, 80, 95, 70],
                name: 'John',
                itemStyle: { color: '#4a90e2' },
                areaStyle: { color: 'rgba(74, 144, 226, 0.3)' }
            },
            {
                value: [90, 85, 95, 75, 80, 85],
                name: 'Sarah',
                itemStyle: { color: '#f39c12' },

7.4 Introduction to Gauge Charts

Gauge charts are perfect for displaying progress, performance metrics, or any value within a defined range. They provide an intuitive visual representation of how a value relates to a target or threshold.

When to Use Gauge Charts:

  • Displaying progress towards a goal
  • Showing performance metrics (KPIs)
  • Representing values within a range
  • Monitoring system status or health

7.5 Basic Gauge Chart

Let's create a simple gauge chart showing a performance metric.

Performance Gauge


var basicGaugeChart = echarts.init(document.getElementById('basic-gauge-chart'));
var basicGaugeOption = {
    title: { text: 'Performance Gauge', left: 'center', textStyle: { color: '#333' } },
    tooltip: {
        formatter: '{a} 
{b} : {c}%'
    },
    series: [{
        name: 'Performance',
        type: 'gauge',
        detail: { formatter: '{value}%' },
        data: [{ value: 75, name: 'Performance' }],
        title: {
            color: '#333',
            fontSize: 16
        },
        axisLine: {
            lineStyle: {
                width: 30,
                color: [
                    [0.3, '#ff6b6b'],
                    [0.7, '#feca57'],
                    [1, '#48dbfb']
                ]
            }
        },
        pointer: {
            itemStyle: {
                color: 'auto'
            }
        },
        axisTick: {
            distance: -30,
            length: 8,
            lineStyle: {
                color: '#fff',
                width: 2
            }
        },
        splitLine: {
            distance: -30,
            length: 30,
            lineStyle: {
                color: '#fff',
                width: 4
            }
        },
        axisLabel: {
            color: 'inherit',
            distance: 40,
            fontSize: 12
        },
        detail: {
            valueAnimation: true,
            formatter: '{value}%',
            color: 'inherit'
        }
    }]
};
basicGaugeChart.setOption(basicGaugeOption);

üí° Pro Tips for Radar & Gauge Charts

  • Radar Charts: Use consistent scales across indicators for fair comparison
  • Gauge Charts: Choose appropriate color schemes (green-yellow-red for performance)
  • Multiple Series: Limit to 3-4 series in radar charts to avoid clutter
  • Animations: Use progressive animations to engage users
  • Responsive Design: Ensure charts work well on all screen sizes

7.8 Key Takeaways

  • Radar charts are excellent for comparing multiple variables across different categories
  • Gauge charts provide intuitive visualization of progress and performance metrics
  • Both chart types support extensive customization for different use cases
  • Use appropriate color schemes and scales to enhance readability
  • Consider the number of data series to avoid visual clutter
